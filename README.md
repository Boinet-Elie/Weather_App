# Weather App

Application météo moderne développée avec .NET MAUI.

## Fonctionnalités

- Affichage des prévisions météo sur 5 jours
- Géolocalisation de la ville
- Interface moderne et épurée
- Page de configuration utilisateur

## Technologies

- .NET MAUI
- C#
- XAML

## API

Utilise l'API [prévision-météo.ch](https://www.prevision-meteo.ch) pour les données météo.

## Lancer le projet

```bash
dotnet restore
dotnet build
dotnet run
```
